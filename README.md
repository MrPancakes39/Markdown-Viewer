## Build From Source

1. Clone the repository
```
$ git clone https://github.com/MrPancakes39/Markdown-Viewer.git
$ cd Markdown-Viewer
```

2. Install dependencies
```
$ yarn
```

3. Build the app
```
$ yarn run gen-tmp
$ yarn run dist
```
